Python Web Services

Welcome to pywebsvcs, a SourceForge "meta-project" for Python web services programming. It supports a range of libraries for both client and server programming, using protocols like SOAP, WSDL, UDDI, etc.

Projects

pywebsvcs currently features the following subprojects:

SOAPpy

SOAP implementation by Cayce Ullman (c_ullman@yahoo.com) and Brian Matthews (blm@blmatthews.com).

ZSI, Zolera SOAP Infrastructure

A different SOAP implementation primarily by Rich Salz (rsalz@zolera.com). Initial home (for 1.0 and 1.1) is http://www.zolera.com/resources/opensrc/zsi, but current development is hosted here at SourceForge.

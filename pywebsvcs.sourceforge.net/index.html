Welcome to the pywebsvcs meta-project.

pywebsvcs provides client-server libraries for web programming using SOAP and related protocols (WSDL, UDDI, etc).

For additional information on Python and XML, check out the Python XML SIG.

Sub-Projects

SOAPpy

A SOAP implementation written by Cayce Ullman and Brian Matthews, now maintained by Gregory Warnes and Ivan R. Judson.

API docs | README file | CVS tree | Latest official release (0.11.5)

ZSI, Zolera SOAP Infrastructure

A different SOAP implementation initially by Rich Salz, now maintained mainly by Rich and Christopher.

API docs | Documentation | CVS tree | Latest official release (1.5)

wstools

A library of WSDL and other classes useful for building Web Services stacks. This is common code used by both SOAPpy and ZSI.

API docs (currently broken) | CVS tree | No separate release

The API docs are updated daily from the CVS repository.
